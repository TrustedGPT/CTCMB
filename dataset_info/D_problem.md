**数据集描述**：D型题（共用选项选择题）与C型题目相同，区别是share_content描述的是每个共用备选答案，subquestion中没有选项。

**数据来源**：中医执业医师考试题目

**元数据**：

```
share_content：被共用的题干
question：问题列表
sub_question：子问题
answer：答案
```

**评估指标**: Accuracy
